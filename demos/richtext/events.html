Events and content access

jQuery.richtext comes with basic event support. Simply listen for richtext-update on the node the constructor was called upon to catch any changes. To access the content of the editor pane, call jQuery.richtext again on the editor node, with the parameter content as first parameter and optionally the new value as second parameter.
