Easy & clean setup and teardown

jQuery.richtext tries to be unintrusive, avoiding such nasty things like wrapper node. Instead everything is put inside the node it's called upon, using it's previous inner HTML as content for the actual editor pane. Similar, on tear down it will put the editor pane's content back into the node as innerHTML.
